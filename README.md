🖼️ Image Converter – PNG→JPG
Мощный конвертер изображений с пакетной обработкой, рекурсивным обходом папок, настройкой качества, изменением размера и сохранением метаданных.
Поддерживает PNG, JPG, BMP, TIFF, WEBP и другие форматы.
Реализован на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Пакетная обработка – конвертация всех изображений в папке за один раз.

Рекурсивный режим – обход всех подпапок (-r).

Настройка качества – от 1 до 100 для JPG (-q, по умолчанию 90).

Изменение размера – масштабирование изображений (-s WIDTHxHEIGHT).

Сохранение метаданных – перенос EXIF-информации (где поддерживается).

Многопоточность – ускорение обработки (в некоторых реализациях).

Прогресс-бар – наглядный индикатор выполнения.

Цветной вывод – информативные сообщения в терминале.

Кроссплатформенность – работает на Windows, Linux, macOS.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <вход> [выход] [опции]
Аргументы
Аргумент	Описание
<вход>	Входной файл или папка с изображениями
[выход]	Выходной файл или папка (если не указан, используется тот же путь)
Опции
Опция	Описание
-q, --quality <N>	Качество JPG (1–100, по умолчанию 90)
-r, --recursive	Рекурсивно обрабатывать подпапки
-s, --size <WxH>	Изменить размер (например, 800x600)
-f, --format <ext>	Выходной формат (по умолчанию jpg)
-v, --verbose	Подробный вывод
-h, --help	Справка
Примеры
bash
# Конвертировать один файл
python converter.py image.png image.jpg

# Конвертировать все PNG в папке в JPG
python converter.py ./images ./output -q 85

# Рекурсивно конвертировать с изменением размера
python converter.py ./photos ./resized -r -s 1024x768

# Конвертировать в WEBP с качеством 80
python converter.py input.png -f webp -q 80

🧠 Логика работы
Сбор файлов – сканирование входной папки (рекурсивно при -r) или использование одного файла.

Фильтрация – только изображения с поддерживаемыми расширениями (PNG, JPG, BMP, TIFF, WEBP).

Загрузка – чтение изображения в память.

Обработка – изменение размера (если указано), конвертация в целевой формат с заданным качеством.

Сохранение – запись в выходной файл с сохранением структуры папок (при рекурсивном режиме).

Многопоточность – в некоторых реализациях для ускорения пакетной обработки.

✨ Дополнительные фичи
Прогресс-бар – отображение процента выполнения.

Сохранение структуры папок – при рекурсивном обходе выходная структура повторяет входную.

Автоопределение формата – если выходной формат не указан, определяется по расширению выходного файла.

Обработка ошибок – программа продолжает работу при ошибках отдельных файлов.
